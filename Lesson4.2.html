Lesson 2: Script Tags

What I learned

So basically for this lesson, we learned the basics to coding Javascript and how all of that worked with the use of the script tag. We learned where to put them in terms of when we want it to load in relation to the websites loading, and if we could have it be external or not to apply it to other pages in our site. Along with that, we learned about the little things like spaces, capitalization, and backslashes in howmuch they matter in the languageof Javascript. Overall, we were taught about how to apply the tool we learned about in Lesson 1 and how to use it. I personally like it alot since it adds more interactivity to a website, making it more engaging, and thus more fun!
