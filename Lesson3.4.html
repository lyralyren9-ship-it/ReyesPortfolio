Lesson 4: Border & Table Formatting

What I learned

So what I learned here was about formatting all the elements like text and images and such with a border, which can be varied in size, width, style, etc. along with how you'd format text into a table; similar to Excel in a way. We'd mainly use it when we have information we have to organize by certain overlapping categories, or something else. Overall, these are important because they're able to implement a common way of organization within a website through coding, making the possibilities of how and where you'd use this practically infinitely flexible.
